# latihan-git
--Folder Latihan | Belajar menggunakan Git &amp; Github


# PERINTAH DASAR GIT

Cek Status
$ git status

Tambahkan semua file ke git
$ git add .

Simpan perubahan dalam git
$ git commit -m "update validasi"

Simpan ke Github
$ git push

Cek remote Github
$ git remote -v

Ambil data guthub (clone)
$ git clone git clone __alamat_github_clone__
